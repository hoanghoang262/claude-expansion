# Routing Decision

## Task
"không biết nên dùng Redis hay MongoDB" (don't know whether to use Redis or MongoDB)

## Phase: Brainstorming

## Rationale

The user is expressing **uncertainty** ("không biết" = don't know) about choosing between two database technologies. This is a classic case for the **Brainstorming** phase, not Research.

### Why NOT Research?
- Research is for: "research", "tìm hiểu", "phân tích" - investigating existing information
- The user hasn't asked to research either technology - they're asking for help DECIDING

### Why Brainstorm?
- User has a **vague idea** (just knows they need a database)
- User expresses **uncertainty** ("không biết" = don't know)
- **Multiple options are possible** (Redis OR MongoDB)

From SKILL.md:
> **Auto-suggest brainstorm** when:
> - User has vague idea
> - User says "not sure", "confused", "what if"
> - Multiple options possible

The phrase "không biết nên dùng" (don't know which to use) directly matches "not sure" criteria.
